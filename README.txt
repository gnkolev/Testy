#Betty Wallet System

BettyWalletSystem is a **console-based betting game** where players can deposit, withdraw, and place bets. It follows a **Clean Architecture** approach with **Dependency Injection, Logging, and Unit Tests**.

##Features
**Deposit & Withdraw Money**  
**Betting System** (Win x2, Win x10, or Lose)  
**Transaction History (Persistent Storage)**  
**Logging System (`wallet_log.txt`)**  
**Unit Tests (xUnit, Moq for mocking services)**  
**Clean Architecture (Domain, Application, Infrastructure, ConsoleApp)**
